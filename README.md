# Flow Engine Web

## 레파지토리 사용 가이드

개발팀은 다음 권고사항을 숙지하여 작업하도록 한다

-   완성되지 않은 소스 및 테스트되지 않은 소스는 등록하지 않는다
-   사용하지 않는 코드 및 개인적인 코드는 등록하지 않는다
    -   사용하지 않거나 변경되어 주석 처리한 코드
    -   콘솔 출력
    -   개인 환경정보
-   프로젝트 팀은 master 브랜치를 사용하지 않는다 (master 브랜치는 개발이 완료되면 관리자가 통합한다)
-   프로젝트 팀은 제공된 브랜치 하에서 사용한다
-   공유작업이 아닌 이상 레파지토리에 브랜치를 생성하지 않는다
-   커밋 시, 명확한 주석을 기록한다
-   개인 작업은 로컬 브랜치를 생성하여 작업하도록 한다
-   원본 브랜치로 소스를 통합할 경우 rebase를 사용한다.
